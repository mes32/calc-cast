# Calc Cast
Server-side calculator that broadcasts calculation history to all users via websockets

## Utilized Web Stack
- `User Interface` - React, CSS
- `Client` - React
- `Server` - Node.js, Express, Socket.io
- `Database` - PostgreSQL

## Requirements
- Git
- Web browser
- Node
- Nodemon
- PostgreSQL
- Heroku CLI (Optional for hosting on Heroku)
